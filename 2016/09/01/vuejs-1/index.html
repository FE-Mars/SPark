什么是 Vue.js？

数据驱动的组件，为现代化的 Web 界面而生

Vue.js的特点

  • 简洁

    HTML 模板 + JSON 数据，再创建一个 Vue 实例，就这么简单。

  • 数据驱动

    自动追踪依赖的模板表达式和计算属性。

  • 组件化

    用解耦、可复用的组件来构造界面。

  • 轻量

    ~24kb min+gzip，无依赖。

  • 快速

    精确有效的异步批量 DOM 更新。

  • 模块友好

    通过 NPM 或 Bower 安装，无缝融入你的工作流。
